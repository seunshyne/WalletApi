# PrimeWallet API

PrimeWallet API is a backend service built with Laravel that powers a digital wallet application. The API handles authentication, wallet management, transactions, and user-related operations.

## Features

* User authentication and authorization
* RESTful API architecture
* Wallet creation and management
* Transaction processing
* Secure API endpoints
* Role-based access control
* Database management and optimization
* API validation and error handling

## Tech Stack

* Laravel (PHP)
* MySQL / PostgreSQL
* Laravel Sanctum
* REST APIs
* Git & GitHub

## Key Responsibilities

Some of the areas I worked on include:

* Building and maintaining API endpoints
* Implementing authentication and authorization
* Database design and query optimization
* API testing and debugging
* Backend business logic implementation
* Integration between frontend and backend services
